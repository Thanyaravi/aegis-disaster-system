QUICK START
═══════════════════════════════════════════════════════════════════════════════

WINDOWS (Easiest):
  1. Double-click: run_all.bat
  2. Wait 10 seconds
  3. Open: http://localhost:5173

MANUAL (Any OS):
  Terminal 1: cd backend && python -m venv venv && activate && pip install -r requirements.txt && uvicorn api.server:app --reload --host 0.0.0.0 --port 8000
  Terminal 2: cd frontend/aegis-ui && npm install && npm run dev
  Terminal 3: cd backend && python main.py
  Browser: http://localhost:5173
