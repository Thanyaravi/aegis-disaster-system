ENDPOINT REFERENCE

METHOD ENDPOINT DESCRIPTION RESPONSE
GET /state Full simulation state — drones, detections, events, alerts JSON: drones[], detections[], events[], alert, step
GET /health System health check for uptime monitoring JSON: status, timestamp
POST /update Push simulation state update from backend loop JSON: ok
POST /sos Manually trigger SOS at grid coordinate pos:[x,y] JSON: triggered, pos
WS /ws Bidirectional WebSocket — real-time state broadcast Stream: state events

DEPLOY AEGIS IN MINUTES

Clone the repo, spin up the backend and frontend — the simulation loop runs automatically. Full local or cloud deployment supported.

QUICKSTART
$ git clone https://github.com/Thanyaravi/aegis-disaster-system
$ cd backend && uvicorn api.server:app --reload
$ cd frontend/aegis-ui && npm run dev
$ python main.py
